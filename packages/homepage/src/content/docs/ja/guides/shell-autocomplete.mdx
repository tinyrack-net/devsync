---
title: シェル補完
description: シェルでdotweaveのタブ補完を設定する方法。
---

import { Aside } from "@astrojs/starlight/components";

タブ補完を使えば、ドキュメントを開かなくてもコマンド、フラグ、引数を素早く確認できます。dotweaveは**Bash**、**Zsh**、**Fish**に対応しています。

## Bash

`~/.bashrc`に以下の行を追加します：

```bash
eval "$(dotweave autocomplete bash)"
```

シェルを再読み込みします：

```bash
source ~/.bashrc
```

## Zsh

`~/.zshrc`に以下の行を追加します：

```bash
eval "$(dotweave autocomplete zsh)"
```

シェルを再読み込みします：

```bash
source ~/.zshrc
```

## Fish

`~/.config/fish/config.fish`に以下の行を追加します：

```fish
dotweave autocomplete fish | source
```

シェルを再読み込みします：

```bash
source ~/.config/fish/config.fish
```

<Aside type="tip">
evalの行を追加したら、新しいターミナルを開く（またはconfigファイルをsourceする）して、
`dotweave `の後にTabキーを押してみてください。利用可能なコマンドの一覧が表示されるはずです。
</Aside>

<Aside>
`volta`や`fnm`などのバージョンマネージャーでdotweaveをインストールした場合は、
evalの行が実行される前に`dotweave`バイナリが`PATH`に含まれていることを確認してください。
</Aside>
