syncディレクトリで新しいシェルを起動します。

`chezmoi cd` と同様に、現在のシェルのディレクトリを変更するのではなく、syncディレクトリをルートとする子シェルを開きます。シェルを終了すると元のセッションに戻ります。

## Usage

```bash
devsync cd
```

<Aside type="tip">
syncディレクトリ内で直接gitコマンド（`commit`、`push`、`pull`）を実行するのに便利です。
</Aside>
