gitベースのsyncディレクトリの内容をローカルの設定パスに適用します。

syncディレクトリからtrackされたアーティファクトを読み込み、ホームディレクトリ配下のローカルパスに展開します。secretアーティファクトは、設定されたage identityで復号されてからローカルに書き込まれます。

## Usage

```bash
dotweave pull [flags]
```

## Flags

| Flag               | Description                          |
| ------------------ | ------------------------------------ |
| `--dry-run`        | 実際には書き込まず、変更内容をプレビューします。     |
| `--profile <name>` | 特定のprofileレイヤーを使用します。        |
| `--verbose`        | 詳細な出力を表示します。                |

## Examples

trackされたすべてのアーティファクトをローカルに適用する:

```bash
dotweave pull
```

ローカルに何が変更されるかプレビューする:

```bash
dotweave pull --dry-run
```

workプロファイルのレイヤーのみpullする:

```bash
dotweave pull --profile work
```

<Aside type="tip">
適用前に何が変更されるか確認するには、先に `dotweave status` を実行してください。
</Aside>
