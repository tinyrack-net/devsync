gitベースのsyncディレクトリを初期化します。

XDGコンフィグディレクトリ配下にdevsyncリポジトリを新規作成、または既存のリポジトリに接続し、以降の `pull` や `push` で使用するsync設定を保存します。リポジトリ引数を省略した場合、syncディレクトリ内にローカルgitリポジトリが作成されます。

## Usage

```bash
devsync init [repository]
```

## Arguments

| Argument       | Description                                              |
| -------------- | -------------------------------------------------------- |
| `[repository]` | cloneするリモートURLまたはローカルgitリポジトリのパスです。省略時はローカルリポジトリが新規作成されます。 |

## Flags

| Flag                        | Description                                                    |
| --------------------------- | -------------------------------------------------------------- |
| `--key <age-private-key>`   | ageの秘密鍵をidentityファイルに保存します。             |
| `--promptKey`               | 自動生成ではなく、ageの秘密鍵を対話的に入力します。 |
| `--verbose`                 | 詳細な出力を表示します。                                          |

## Examples

新しいローカルsyncディレクトリを作成する:

```bash
devsync init
```

リモートリポジトリをsyncディレクトリとしてcloneする:

```bash
devsync init https://github.com/user/dotfiles.git
```

既存のageの秘密鍵を指定して初期化する:

```bash
devsync init --key "AGE-SECRET-KEY-1..."
```

<Aside type="tip">
別のマシンで使用しているage identityがある場合は、`--key` または `--promptKey` で指定すると、secretファイルを復号できます。
</Aside>
