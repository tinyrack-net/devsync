description: "パスの同期トラッキングを停止します。"
---

同期対象のパスのトラッキングを停止します。

trackされたルートエントリまたはネストされたオーバーライドをdevsyncの設定から削除します。この操作は設定の更新のみを行い、実際のファイル変更は次回の `push` または `pull` で反映されます。ローカルパスを指定するとメインのtrackターゲットが削除され、trackされたディレクトリ内のリポジトリ相対の子パスを指定するとそのオーバーライドのみが削除されます。

## Usage

```bash
devsync untrack <target> [flags]
```

## Arguments

| Argument   | Description                                             |
| ---------- | ------------------------------------------------------- |
| `<target>` | トラッキングを停止するローカルパスまたはリポジトリパスです。 |

## Flags

| Flag        | Description            |
| ----------- | ---------------------- |
| `--verbose` | 詳細な出力を表示します。  |

## Examples

設定ファイルのトラッキングを停止する:

```bash
devsync untrack ~/.config/starship.toml
```

trackされたディレクトリ内の子オーバーライドを削除する:

```bash
devsync untrack .ssh/config
```

<Aside>
`untrack` は `manifest.jsonc` を更新するだけです。ローカルファイルやsyncディレクトリのアーティファクトは削除**しません**。
</Aside>
