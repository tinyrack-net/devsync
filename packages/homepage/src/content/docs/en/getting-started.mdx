- **Node.js 24** or later

## Installation

```bash
npm install -g @tinyrack/devsync
```

## Starting Fresh

If this is your first time using devsync, follow these steps to create a new sync directory.

<Steps>

1. **Initialize**

   Create a new sync directory with a fresh age key for encrypting secrets.

   ```bash
   devsync init
   ```

   This sets up `~/.config/devsync/` with a Git repository and an age identity.

2. **Track your files**

   Tell devsync which files to sync. For example:

   ```bash
   devsync track ~/.zshrc
   devsync track ~/.config/starship.toml
   devsync track ~/.config/nvim
   ```

   For sensitive files like SSH keys, use secret mode:

   ```bash
   devsync track --mode secret ~/.ssh/config
   ```

3. **Push to the sync directory**

   Copy your tracked files into the sync directory:

   ```bash
   devsync push
   ```

4. **Commit and push with Git**

   Open the sync directory and push to a remote:

   ```bash
   devsync cd
   git add -A && git commit -m "Initial sync"
   git remote add origin <your-repo-url>
   git push -u origin main
   ```

</Steps>

<Aside type="tip">
  After setting up, check out the [Directory Structure](/en/guides/directory-structure/) guide to see where devsync stores your config and synced files.
</Aside>

## Connecting an Existing Sync Directory

On a new machine, clone your existing sync directory:

<Steps>

1. **Clone and initialize**

   ```bash
   devsync init <your-repo-url>
   ```

   This clones the repository into `~/.config/devsync/repository/` and sets up a new age identity. If you already have an age private key from another machine, pass it with `--key`:

   ```bash
   devsync init <your-repo-url> --key "AGE-SECRET-KEY-1..."
   ```

2. **Pull your files**

   Restore all tracked files to their local paths:

   ```bash
   devsync pull
   ```

</Steps>

<Aside type="note">
  If your sync directory contains encrypted secrets, you need the same age private key on all machines. See [Syncing Secrets](/en/guides/syncing-secrets/) for details.
</Aside>
