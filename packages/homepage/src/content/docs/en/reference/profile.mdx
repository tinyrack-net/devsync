---
title: "dotweave profile"
description: "Manage sync profiles."
---

Manage active and assigned sync profiles.

Inspect configured profiles or update which profile dotweave should use by default.

## Subcommands

### `dotweave profile list`

Show configured and active profiles.

```bash
dotweave profile list
```

### `dotweave profile use [name]`

Set or clear the active profile. When called without a name, the active profile is cleared.

```bash
dotweave profile use [name]
```

## Examples

List all configured profiles:

```bash
dotweave profile list
```

Activate the work profile:

```bash
dotweave profile use work
```

Clear the active profile:

```bash
dotweave profile use
```
