## 설치

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install -g @tinyrack/devsync
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add -g @tinyrack/devsync
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn global add @tinyrack/devsync
    ```
  </TabItem>
</Tabs>

설치 후 `devsync --version`으로 정상적으로 설치됐는지 확인해요.

## 새로운 저장소로 시작하기

<Steps>

1. **저장소 초기화**

   ```bash
   devsync init
   ```

   실행하면 age 개인키 입력 여부를 물어봐요. 처음 시작이라면 그냥 엔터를 누르면 새 키를 자동으로 생성해요.

   초기화가 완료되면 `~/.config/devsync/` 아래에 다음 구조가 만들어져요.

   <FileTree>
   - ~/.config/devsync/
     - sync/          Git 저장소 (동기화 파일이 저장되는 곳)
     - age/
       - keys.txt     age 개인키 (시크릿 파일 복호화에 사용)
     - settings.json  전역 설정 (활성 프로파일 등)
   </FileTree>

   <Aside type="caution">
   `age/keys.txt`는 시크릿 파일을 복호화하는 데 사용하는 개인키예요. 이 파일을 잃어버리면 암호화된 파일을 복원할 수 없으니 **안전한 곳에 별도로 백업해 두세요.**
   </Aside>

2. **파일 추적 등록**

   동기화할 파일이나 디렉터리를 `track` 명령으로 등록해요.

   ```bash
   devsync track ~/.gitconfig
   devsync track ~/.zshrc
   devsync track ~/.config/nvim
   ```

   민감한 파일은 `--mode secret`으로 등록하면 저장소에 암호화되어 저장돼요.

   ```bash
   devsync track ~/.ssh/config --mode secret
   ```

3. **저장소에 반영**

   ```bash
   devsync push
   ```

   로컬 파일들을 `~/.config/devsync/sync/` 안에 미러링해요. 이 시점에는 아직 git commit이 만들어지지 않아요.

4. **Git 원격 저장소에 올리기**

   `devsync cd`로 sync 저장소 디렉터리 안으로 들어가서 직접 git 명령을 실행해요.

   ```bash
   devsync cd
   ```

   ```bash
   git remote add origin https://github.com/yourname/dotfiles.git
   git add .
   git commit -m "Initial sync"
   git push -u origin main
   exit
   ```

</Steps>

## 기존 저장소 연동하기

새 기기에서 이미 만들어둔 저장소를 불러올 때는 `init`에 저장소 URL을 넘겨요.

```bash
devsync init https://github.com/yourname/dotfiles.git
```

클론이 완료되면 age 개인키를 입력하라는 메시지가 나와요. 기존에 백업해 둔 키를 그대로 입력하면 돼요.

<Aside>
age 개인키는 `AGE-SECRET-KEY-` 로 시작하는 문자열이에요. 키를 파일로 가지고 있다면 `--key` 옵션으로 직접 넘길 수도 있어요.

```bash
devsync init https://github.com/yourname/dotfiles.git --key AGE-SECRET-KEY-...
```
</Aside>

이후 파일을 원래 경로에 복원해요.

```bash
devsync pull
```

시크릿 파일은 age 키로 자동 복호화되어 저장돼요.
