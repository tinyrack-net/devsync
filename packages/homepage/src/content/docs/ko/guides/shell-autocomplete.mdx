탭 자동완성을 사용하면 문서를 찾지 않아도 명령어, 플래그, 인자를 발견할 수
있어요. devsync는 **Bash**, **Zsh**, **Fish**를 지원해요.

## Bash

`~/.bashrc`에 이 줄을 추가하세요:

```bash
eval "$(devsync autocomplete bash)"
```

그런 다음 셸을 다시 로드하세요:

```bash
source ~/.bashrc
```

## Zsh

`~/.zshrc`에 이 줄을 추가하세요:

```bash
eval "$(devsync autocomplete zsh)"
```

그런 다음 셸을 다시 로드하세요:

```bash
source ~/.zshrc
```

## Fish

`~/.config/fish/config.fish`에 이 줄을 추가하세요:

```fish
devsync autocomplete fish | source
```

그런 다음 셸을 다시 로드하세요:

```bash
source ~/.config/fish/config.fish
```

<Aside type="tip">
eval 줄을 추가한 후, 새 터미널을 열거나 설정 파일을 source한 뒤 `devsync `를
입력하고 Tab을 눌러 보세요. 사용 가능한 명령어 목록이 나타나야 해요.
</Aside>

<Aside>
`volta`나 `fnm` 같은 버전 관리자로 devsync를 설치했다면, eval 줄이 실행되기
전에 `devsync` 바이너리가 `PATH`에 포함되어 있는지 확인하세요.
</Aside>
