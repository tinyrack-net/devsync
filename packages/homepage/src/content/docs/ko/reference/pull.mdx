git 기반 sync 디렉터리의 내용을 로컬 설정 경로에 적용해요.

sync 디렉터리에서 추적 중인 파일을 읽어 홈 디렉터리 아래 로컬 경로에 복원해요. secret 파일은 설정된 age identity로 복호화한 뒤 로컬에 기록해요.

## Usage

```bash
devsync pull [flags]
```

## Flags

| Flag               | Description                          |
| ------------------ | ------------------------------------ |
| `--dry-run`        | 실제로 쓰지 않고 변경 내용을 미리 확인해요.     |
| `--profile <name>` | 특정 profile 레이어를 사용해요.        |
| `--verbose`        | 상세한 출력을 보여줘요.                |

## Examples

추적 중인 모든 파일을 로컬 경로에 적용해요:

```bash
devsync pull
```

로컬에 어떤 변경이 생길지 미리 확인해요:

```bash
devsync pull --dry-run
```

work profile 레이어만 적용해요:

```bash
devsync pull --profile work
```

<Aside type="tip">
적용 전에 `devsync status`를 먼저 실행하면 `pull`이 바꿀 내용을 미리 확인할 수 있어요.
</Aside>
