---
title: "devsync profile"
description: "sync 프로필을 관리해요."
---

활성 profile과 할당된 sync profile을 관리해요.

설정된 profile을 확인하거나, devsync가 기본으로 사용할 profile을 변경할 수 있어요.

## Subcommands

### `devsync profile list`

설정된 profile과 활성 profile을 보여줘요.

```bash
devsync profile list
```

### `devsync profile use [name]`

활성 profile을 설정하거나 해제해요. 이름 없이 실행하면 활성 profile이 해제돼요.

```bash
devsync profile use [name]
```

## Examples

설정된 모든 profile을 확인해요:

```bash
devsync profile list
```

work profile을 활성화해요:

```bash
devsync profile use work
```

활성 profile을 해제해요:

```bash
devsync profile use
```
