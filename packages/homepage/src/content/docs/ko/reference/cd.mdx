sync 디렉터리에서 셸을 열어요.

`chezmoi cd`처럼, 현재 셸의 디렉터리를 바꾸는 대신 로컬 sync 디렉터리를 루트로 하는 새 자식 셸을 열어요. 셸을 종료하면 이전 세션으로 돌아가요.

## Usage

```bash
dotweave cd
```

<Aside type="tip">
sync 디렉터리 안에서 git 명령(`commit`, `push`, `pull`)을 직접 실행할 때 유용해요.
</Aside>
