현재 sync 설정에 대해 push와 pull이 변경할 내용을 보여줘요.

추적 중인 로컬 파일과 sync 디렉터리를 비교해서, `push`가 저장소에 쓸 내용과 `pull`이 로컬에 적용할 내용을 알려줘요.

## Usage

```bash
dotweave status [flags]
```

## Flags

| Flag               | Description                          |
| ------------------ | ------------------------------------ |
| `--profile <name>` | 특정 profile 레이어를 사용해요.        |
| `--verbose`        | 상세한 출력을 보여줘요.                |

## Examples

전체 sync 상태를 확인해요:

```bash
dotweave status
```

특정 profile의 상태를 확인해요:

```bash
dotweave status --profile work
```

<Aside>
`status`는 `manifest.jsonc`를 읽고 파일 내용을 비교해요. `git status`를 실행하는 것이 **아니라**, dotweave 수준의 차이만 보여줘요.
</Aside>
