추적 중인 경로의 동기화를 중단해요.

devsync 설정에서 추적 중인 루트 항목이나 하위 재정의를 제거해요. 이 명령은 설정만 갱신하고, 실제 파일 변경은 다음 `push`나 `pull`에서 이루어져요. 로컬 경로를 지정하면 추적 대상 자체를 제거하고, 추적 중인 디렉터리 안의 저장소 상대 경로를 지정하면 해당 재정의만 제거해요.

## Usage

```bash
devsync untrack <target> [flags]
```

## Arguments

| Argument   | Description                                             |
| ---------- | ------------------------------------------------------- |
| `<target>` | 추적을 중단할 로컬 경로 또는 저장소 경로예요. |

## Flags

| Flag        | Description            |
| ----------- | ---------------------- |
| `--verbose` | 상세한 출력을 보여줘요.  |

## Examples

설정 파일의 추적을 중단해요:

```bash
devsync untrack ~/.config/starship.toml
```

추적 중인 디렉터리 안의 자식 재정의를 제거해요:

```bash
devsync untrack .ssh/config
```

<Aside>
`untrack`은 `manifest.json`만 갱신해요. 로컬 파일이나 sync 디렉터리의 파일을 **삭제하지 않아요**.
</Aside>
