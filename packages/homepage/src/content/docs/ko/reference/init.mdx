git 기반 sync 디렉터리를 초기화해요.

XDG config 디렉터리 아래에 로컬 devsync 저장소를 만들거나 연결한 뒤, 이후 `pull`과 `push`에서 사용할 sync 설정을 저장해요. repository 인자를 생략하면 sync 디렉터리에 로컬 git 저장소를 새로 만들어요.

## Usage

```bash
devsync init [repository]
```

## Arguments

| Argument       | Description                                              |
| -------------- | -------------------------------------------------------- |
| `[repository]` | clone할 원격 URL 또는 로컬 git 저장소 경로예요. 생략하면 새 로컬 저장소를 만들어요. |

## Flags

| Flag                        | Description                                                    |
| --------------------------- | -------------------------------------------------------------- |
| `--key <age-private-key>`   | age 개인 키를 identity 파일에 저장해요.             |
| `--promptKey`               | age 개인 키를 자동 생성하지 않고 직접 입력받아요. |
| `--verbose`                 | 상세한 출력을 보여줘요.                                          |

## Examples

새 로컬 sync 디렉터리를 초기화해요:

```bash
devsync init
```

원격 저장소를 clone해서 sync 디렉터리로 사용해요:

```bash
devsync init https://github.com/user/dotfiles.git
```

기존 age secret key로 초기화해요:

```bash
devsync init --key "AGE-SECRET-KEY-1..."
```

<Aside type="tip">
다른 머신에서 쓰던 age identity가 있다면 `--key`나 `--promptKey`로 전달하세요. secret 파일을 복호화할 수 있어요.
</Aside>
