셸 자동완성 스크립트를 출력해요.

eval 기반 셸 설정에서 사용할 수 있는 셸별 자동완성 스크립트를 생성해요.

## Usage

```bash
devsync autocomplete <shell>
```

## Subcommands

| Subcommand                  | Description                    |
| --------------------------- | ------------------------------ |
| `devsync autocomplete bash` | Bash용 자동완성 스크립트를 출력해요. |
| `devsync autocomplete zsh`  | Zsh용 자동완성 스크립트를 출력해요.  |
| `devsync autocomplete fish` | Fish용 자동완성 스크립트를 출력해요. |

## Examples

**Bash**에서 자동완성을 설정해요 (`~/.bashrc`에 추가):

```bash
eval "$(devsync autocomplete bash)"
```

**Zsh**에서 자동완성을 설정해요 (`~/.zshrc`에 추가):

```bash
eval "$(devsync autocomplete zsh)"
```

**Fish**에서 자동완성을 설정해요 (`~/.config/fish/config.fish`에 추가):

```fish
devsync autocomplete fish | source
```
